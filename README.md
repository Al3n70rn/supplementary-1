3. [Zuguang Gu, Roland Eils and Matthias Schlesner. gtrellis: an R/Bioconductor package for making genome-level Trellis graphics, 2016. BMC Bioinformatics.](http://jokergoo.github.com/supplementary/gtrellis/)
